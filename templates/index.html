Predictive Analysis: Classification of Glass

The following form is part of my first end to end machine learning project ever. The aim of this project is to classify the type of glass based on the refractive index as well as the proportion of each of it's constituent elements. Although this data would usually be impractical to get, the project is pretty much just for the sake of learning and practicing.


The data used for training and testing the model was obtained from the Glass Classification by UCI Machine Learning dataset on Kaggle.


The classification algorithm used here is the XGBoost Classification Algorithm and, after testing, I found the accuracy to be around 93%
